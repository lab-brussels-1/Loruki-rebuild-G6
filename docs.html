Introduction

We can help you with everything related to the cloud hosting and web development as you require. We have a pretty user-friendly interface and easy installation that will help you start running since day number 1.

This is an easy demonstration of the installation part.

Easy access everywhere.

With Loruki, you can access our services and products from pretty much every devices you have with internet connection.

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install loruki-cli

NPM

$ npm install loruki-cli

Yarn

$ yarn install loruki-cli
